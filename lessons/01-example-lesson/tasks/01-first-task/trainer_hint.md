## Trainer Notes

### Common Issues

1. **Windows users**: May need to run terminal as Administrator
2. **Mac users**: May need to install Xcode Command Line Tools first
3. **Linux users**: May need to install build-essential

### Time Estimate

- Fast learners: 5 minutes
- Average: 10 minutes
- With troubleshooting: 15-20 minutes

### Discussion Points

- Why do we use npm/pnpm/yarn?
- What is the purpose of package-lock.json?
- How does the dev server hot-reload work?
